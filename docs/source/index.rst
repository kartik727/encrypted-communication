Welcome to Encrypted Communication's documentation!
===================================================

**Encrypted Communication** is a Python library for demonstrating basic 
techniques to ensure secure and reliable communication between two parties.

Check out the :doc:`usage` section for further information, including
:ref:`installation` instructions.

.. note::

   This project is under active development.
